1. Basic Widgets

Container → customizable box (color, padding, margin, border, size)

Padding → adds padding around a child

Align → aligns child within parent

Center → centers a child

SizedBox → fixed size box, spacer

FittedBox → scales child to fit

AspectRatio → maintains width/height ratio

FractionallySizedBox → size relative to parent

Expanded → fills available space in Row/Column

Spacer → flexible empty space in Row/Column

ClipRRect, ClipOval, ClipPath → clipping shapes

DecoratedBox → custom decoration

2. Text & Icon Widgets

Text → display text

RichText → styled spans

isSelectableText → text that can be selected

Icon → material icons

Image → asset, network, memory

CircleAvatar → circular image/avatar

TextButton, ElevatedButton, OutlinedButton → buttons

IconButton → tappable icon

FloatingActionButton → circular action button

3. Layout Widgets

Row → horizontal layout

Column → vertical layout

Stack → overlapping children

Positioned → position child inside stack

Wrap → auto wrap children in multiple lines

Flow → advanced custom layout

ListView → scrollable vertical/horizontal list

GridView → scrollable grid

SingleChildScrollView → scrollable when content overflows

CustomScrollView → slivers

SliverList, SliverGrid → inside CustomScrollView

4. Input & Form Widgets

TextField → single-line or multi-line text input

TextFormField → with validation

Form → group multiple fields

Checkbox, CheckboxListTile

Radio, RadioListTile

Switch, SwitchListTile

Slider → continuous values

RangeSlider → min/max range

DropdownButton → select from list

PopupMenuButton → context menu

Stepper → stepper UI for multi-step forms

DatePicker, TimePicker → dialogs

5. Scrollable & List Widgets

ListView.builder → lazy loading list

ListView.separated → list with separators

ListTile → standard list item (leading, title, subtitle, trailing)

ReorderableListView → drag & drop list

GridView.builder → lazy grid

PageView → horizontal paging

ScrollView → generic scrollable area

Scrollbar → scroll indicator

RefreshIndicator → pull-to-refresh

6. Containers & Decorations

Card → Material card

Chip → pill-shaped info

ChoiceChip → isSelectable chip

ListTile → reusable row for lists

Drawer → side navigation menu

BottomNavigationBar → bottom tabs

TabBar / TabBarView → top tabs

AnimatedOpacity, AnimatedAlign, AnimatedPositioned

Hero → shared element transition

9. Material Components

Chip, InputChip, ChoiceChip, FilterChip, ActionChip

10. Utility & Helpers

FutureBuilder → async data builder

StreamBuilder → stream data builder

LayoutBuilder → responsive layouts

MediaQuery → screen info

Theme → access app theme
