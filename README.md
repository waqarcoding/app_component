# app_component

A lightweight Flutter package that provides reusable and customizable chip widgets for selection-based UI.

## Features

- ✅ Simple `ChipWidget` using Flutter `ChoiceChip`
- ✅ Advanced `FlutterChipWidget` with optional radio-style selection
- ✅ Horizontal scrolling support
- ✅ Generic type support (`<T>`)
- ✅ Fully Flutter-native (no custom assets or dependencies)
- ✅ Works with light & dark themes
- ✅ pub.dev compatible

---

## Installation

Add this to your `pubspec.yaml`:

```yaml
dependencies:
  app_component: ^0.1.0
